You will be randomly assigned one Java source code file at a time. You must submit a response for this file before you can submit a response for another.

At minimum, you will need to identify/specify the following:

  • The design pattern used in the source code file.
  • Your confidence level in selecting the said design pattern.
  • A 2-3 sentence summary of what the source code in the current file does.

Optionally, you may also include explanations regarding your decisions for the above.

Important:
If the pattern is not listed, please select "~Other Pattern~" and type in the name of the pattern into the pattern explanation field directly below.
If the design pattern identified also spans across or involves several other files, please include these findings in the summary section.

You may freely browse any other file in the same project as the currently assigned file, especially if it has other dependencies within the same project. You may also browse your previous responses and edit them at any time.
